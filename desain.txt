- Nama file gambar:  
  - `gray_shoe.jpg`  
  - `green_shoe.jpg`  
  - `maroon_shoe.jpg`

  - `white_shoe.jpg`

> Gunakan `<img src="Asset/img/nama_file.jpg" class="img-fluid" alt="...">` di kode HTML nanti.

---

## 🧭 **ALUR PEMBELIAN USER**

1. User **mendaftar** → login → otomatis redirect ke halaman utama.
2. Pilih produk → klik **“Tambah ke Keranjang”**.
3. Buka **Keranjang** → review item → klik **“Checkout”**.
4. Isi/cek alamat → pilih metode bayar → **konfirmasi pesanan**.
5. Pesanan masuk ke sistem → admin memproses → user bisa cek status di **Riwayat Pesanan**.

---

## ✅ **CATATAN TAMBAHAN**

- Gunakan Bootstrap 5 untuk responsive dan komponen UI (navbar, card, button, form, dll).
- Untuk login, buat halaman sederhana dengan form email & password.
- Admin hanya bisa diakses jika user memiliki role “admin” (login terpisah atau cek session role).
- Semua halaman wajib include `header.php`, `navbar.php`, dan `footer.php` untuk konsistensi tampilan.S
